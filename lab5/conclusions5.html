Висновки до лабораторної роботи

Під час виконання цієї лабораторної роботи я здобув практичні навички роботи з основними конструкціями мови JavaScript, зокрема з умовними операторами, циклами, масивами та функціями. Я також навчився використовувати JavaScript для динамічного оновлення елементів сторінки та виконання логічних операцій для вирішення різноманітних завдань.
